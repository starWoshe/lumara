## Опис затверджених образів

### Портрет (umbra-portrait-1.png)
- Вік: ~45 років, попелясто-сиве волосся пряме, прибране назад
- Очі: крижано-сині/сірі, пронизливий погляд
- Бліда шкіра, без емоцій на обличчі
- Чорний облягаючий лонгслів + темно-сіра довга спідниця (мінімалізм)
- Тримає срібне ручне дзеркало лівою рукою
- MORA (біла миша) на правому плечі
- Стоїть перед великим дзеркалом у чорній рамці
- Холодне синє підсвічування з вікна, чорна свічка на підставці

### Кімната (umbra-room.png)
- Готичне склепіння, майже порожня кімната
- Велике підлогове дзеркало в орнаментованій рамці — центр
- Одна чорна свічка на залізній підставці
- Шкіряне крісло в кутку
- Стіна з вирізьбленими езотеричними символами (алхімія, астрологія)
- Готичне вікно — холодне синє світло галактики
- MORA (біла миша) видна на підлозі і біля крісла
- Прозорий фон (PNG з альфа-каналом) ✅
- Колірна палітра: чистий чорний, холодне срібло, крижано-блакитний
