## Опис затверджених образів

### Портрет (arcas-portrait-1.png)
- Вік: ~50 років, темне волосся з сивиною на скронях
- Коротка борода з сивиною
- Темно-карі очі, тепла усмішка
- Бордово-фіолетовий оксамит з золотою вишивкою
- Перстень з темним каменем (гранат) — на правій руці
- Тримає карту The Moon обома руками
- На столі: карти таро розкладені, аметист, чорний обсидіан, свічки

### Кімната (arcas-room.png)
- Готичний склепінчастий стелю з рунами
- Книжкова полиця ліворуч: книги, пляшки, шкатулки
- Центр: дзеркало в орнаментованій рамці (змії + місяць)
- Стіл: карти таро, червоні свічки, чорні кристали, сувої
- Вікно праворуч: темно-червоні штори, ворон на підставці, галактика назовні
- Люстра: червоні свічки
- Колірна палітра: темно-коричневий, бордо, червоні свічки ✅
