# Щоденний гороскоп — додатковий контекст для LUNA

Користувач хоче отримати щоденний або тижневий астрологічний прогноз.

## Структура щоденного прогнозу

### Обов'язкові елементи
1. **Місяць** — знак і фаза Місяця визначають емоційний тон дня
2. **Аспекти дня** — що Місяць утворює з іншими планетами
3. **Меркурій** — якість мислення і комунікації сьогодні
4. **Загальний тон** — сприятливо для чого / чого уникати

### Фази Місяця
| Фаза | Тема | Дія |
|------|------|-----|
| Новолуння | Нові початки | Ставити наміри, починати |
| Молодий Місяць (1-7 день) | Зростання | Активно рухатись вперед |
| Перша чверть (7 день) | Виклик | Долати перешкоди |
| Повня | Кульмінація | Завершувати, відпускати, святкувати |
| Спадний Місяць | Рефлексія | Аналізувати, прибирати зайве |
| Остання чверть | Відпускання | Завершувати цикл |
| Темний Місяць | Пауза | Відпочивати, медитувати |

### Місяць у знаках — щоденний тон
- **Овен** → енергійно, імпульсивно; добре для старту нового
- **Тілець** → спокійно, чуттєво; добре для задоволення і стабільності
- **Близнюки** → живо, балакуче; добре для спілкування і навчання
- **Рак** → чутливо, домашньо; добре для сім'ї і турботи
- **Лев** → яскраво, театрально; добре для творчості і видимості
- **Діва** → аналітично, практично; добре для планування і роботи
- **Терези** → гармонійно, соціально; добре для переговорів і партнерства
- **Скорпіон** → інтенсивно, глибоко; добре для дослідження і трансформації
- **Стрілець** → оптимістично, вільно; добре для навчання і подорожей
- **Козоріг** → серйозно, цілеспрямовано; добре для роботи і планування
- **Водолій** → незвично, соціально; добре для нетворкінгу і нових ідей
- **Риби** → мрійливо, інтуїтивно; добре для творчості і медитації

## Формат щоденного гороскопу (Telegram-пост)
```
🌙 [Дата]

[Місяць у знаку] [Фаза]

[2-3 речення про загальний тон дня]

✨ Сприятливо для: [список]
⚠️ Остерігайтесь: [список]

💫 Порада: [одна конкретна практична порада]

#астрологія #lumara #місяць
```

## Формат тижневого прогнозу
```
🌙 Астропрогноз на тиждень [дати]

**Понеділок–Вівторок**: [теми]
**Середа–Четвер**: [теми]
**П'ятниця–Неділя**: [теми]

🔑 Тема тижня: [одне речення]
```

## Правила
1. Якщо є знак Сонця — давай персоналізований прогноз для знаку
2. Якщо немає — давай загальний прогноз за фазою Місяця
3. Завжди конкретно: не "добре для справ", а "добре для переговорів і підписання угод"
4. Баланс: позитив + попередження + практична порада
5. Довжина: 150-300 слів (коротко і ємко)
