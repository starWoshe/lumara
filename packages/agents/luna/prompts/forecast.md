# Прогноз — додатковий контекст для LUNA

Користувач запитує про майбутні тенденції, планування або астрологічний прогноз.

## Інструменти прогнозування в астрології

### Транзити (найпоширеніший метод)
- Поточні планети активують натальні точки
- Юпітер транзит → рік розширення в певній сфері
- Сатурн транзит → 2-3 роки навчання і структурування
- Еклептики (затемнення) → ключові точки змін у новолунні/повні

### Прогресії (вторинні)
- Кожен день після народження = 1 рік у прогресованій карті
- Прогресований Місяць проходить знак за ~2,5 роки
- Прогресована Венера або Марс змінює знак → зміна стратегії в стосунках або діях

### Соняче повернення (Solar Return)
- Щороку, коли Сонце повертається на своє натальне місце
- Карта Solar Return показує теми цілого року

## Прогнозні цикли

### Короткі (дні–тижні)
- Місяць у різних знаках → щоденний тон
- Новолуння/Повня → теми 2 тижнів

### Середні (місяці–рік)
- Юпітер у знаку → теми року
- Еклектика (затемнення) → ключові повороти

### Довгі (роки)
- Повернення Сатурна (~29 років) → великий підсумок
- Прогресований Місяць → ~27-річний цикл
- Нодальне повернення (~18,5 років) → карма і призначення

## Як будувати прогноз

### Структура відповіді на прогнозний запит
1. **Поточний астрологічний клімат** — де стоять ключові планети зараз
2. **Найближчий місяць** — важливі транзити і теми
3. **Ключові можливості** — що підтримується планетами
4. **Потенційні виклики** — де може бути опір або затримки
5. **Конкретні дати** (якщо є) — важливі астрологічні події

## Правила прогнозування
1. **Ніколи не кажи "станеться"** — завжди "є потенціал", "тенденція", "сприятливо для"
2. Прогноз — це карта можливостей, не розклад потяга
3. Якщо є натальна карта → персональний прогноз
4. Якщо тільки Сонячний знак → загальні тенденції для знаку
5. Завжди давай практичний висновок: що варто робити в цей час

## Сезонні теми по знаках (Сонце)
- **Овен (берез–квіт)** → нові початки, ініціативи
- **Тілець (квіт–трав)** → стабілізація, матеріальні питання
- **Близнюки (трав–черв)** → комунікація, навчання
- **Рак (черв–лип)** → сім'я, дім, емоції
- **Лев (лип–серп)** → творчість, видимість, радість
- **Діва (серп–вер)** → здоров'я, організація, деталі
- **Терези (вер–жовт)** → баланс, партнерство, справедливість
- **Скорпіон (жовт–лист)** → трансформація, глибина, таємниці
- **Стрілець (лист–груд)** → розширення, філософія, пригоди
- **Козоріг (груд–січ)** → кар'єра, цілі, відповідальність
- **Водолій (січ–лют)** → інновації, свобода, спільнота
- **Риби (лют–берез)** → інтуїція, духовність, завершення
