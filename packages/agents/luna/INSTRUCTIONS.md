# INSTRUCTIONS.md — Агент LUNA

## 1. Призначення

LUNA — астрологічний AI-провідник LUMARA Academy. Агент аналізує натальні карти, транзити планет, синастрію та надає щоденні астрологічні прогнози. Щодня о 09:00 за Київським часом публікує астрологічний пост у Telegram-канал.

---

## 2. Технічний стек

| Компонент | Технологія |
|-----------|-----------|
| AI-модель | Claude Sonnet 4.6 (Anthropic) |
| Мова (основна) | TypeScript |
| Мова (скрипти) | Python 3.11+ |
| Астрономія | `ephem` (PyEphem) |
| Telegram | `python-telegram-bot` або `httpx` (Bot API) |
| Планувальник | GitHub Actions cron |
| Зберігання | Supabase PostgreSQL (через Prisma) |

---

## 3. Структура модуля

```
packages/agents/luna/
  INSTRUCTIONS.md         ← цей файл
  index.ts                ← точка входу, клас LunaAgent
  personality.ts          ← характер, голос, шаблони відповідей
  prompts/
    natal.md              ← промпт для натальної карти
    transits.md           ← промпт для транзитів планет
    synastry.md           ← промпт для синастрії
    forecast.md           ← промпт для прогнозу
    daily.md              ← промпт для щоденного гороскопу
  scripts/
    calendar.py           ← обчислення астрономічних подій (30 днів)
    telegram_bot.py       ← публікація в Telegram
    requirements.txt      ← залежності Python
```

---

## 4. Правила поведінки LUNA

### Голос і тон
- Містичний, але не пихатий
- Поєднує точну астрологію з практичними порадами
- Мова — автоматичне визначення (UA/EN/RU)
- Ніколи не дає категоричних передбачень
- Завжди підкреслює вільну волю людини

### Типи запитів та відповідні промпти
| Ключові слова | Промпт |
|---------------|--------|
| "натальна карта", "натал", "natal" | natal.md |
| "транзит", "сьогодні", "зараз", "transit" | transits.md |
| "сумісність", "партнер", "синастрія", "synastry" | synastry.md |
| "прогноз", "майбутнє", "forecast" | forecast.md |
| "день", "тиждень", "щоденний", "daily" | daily.md |
| Все інше | базовий системний промпт |

### Формат відповіді
1. Короткий астрологічний контекст (1-2 речення)
2. Конкретна інтерпретація
3. Практична порада або питання для роздумів

## 5. Інтеграції

- **`packages/agents/src/index.ts`** — реєструє LUNA в AGENT_PROMPTS
- **`packages/database`** — зберігає сесії (conversations, messages)
- **`apps/web/api/chat`** — використовує LUNA через HTTP POST
- **Telegram Bot API** — щоденна публікація астрологічних постів
- **GitHub Actions** — cron щодня о 07:00 UTC (09:00 Kyiv)

---

## 6. Змінні середовища

```env
# Обов'язкові
ANTHROPIC_API_KEY=sk-ant-...

# Для Telegram бота
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHANNEL_ID=@lumara_astro  # або числовий ID каналу

# Опційні (для детальніших натальних карт)
ASTRO_TIMEZONE=Europe/Kyiv
```

## 8. Порядок запуску скриптів

```bash

cd packages/agents/luna/scripts
pip install -r requirements.txt

# Тест обчислення календаря
python calendar.py

# Тест відправки в Telegram (потрібен TELEGRAM_BOT_TOKEN)

```
