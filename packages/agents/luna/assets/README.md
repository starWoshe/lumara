## Ключові риси образу (НЕЗМІННО)

- Темне довге волосся з легкими сріблястими відблисками
- **Глибокі блакитні очі** — ключова риса, завжди видно
- Світла природна шкіра, вік 35-40
- Темний оксамит з вишивкою (зірки, місяці, сузір'я)
- Срібний кулон-серп місяця — обов'язково
- Срібні сережки (місяць або зірки)
